# ChatBot 插件

这是一个运行在 **Spigot** 平台上的 ChatBot 插件，旨在为 Minecraft 服务器提供智能聊天功能。

## 特性
- 支持 AI 自动回复玩家消息
- 可配置的消息过滤规则
- 自定义上下文历史记录长度
- 高效异步处理，减少对服务器性能的影响

## 当前状态
**开发中** 🚧  
本插件仍在开发阶段，功能可能不完整，且存在潜在问题。

## 使用方法
1. 将插件 `.jar` 文件放入服务器的 `plugins` 文件夹。
2. 启动服务器，生成默认配置文件。
3. 根据需要修改配置文件，重启服务器以应用更改。

## 依赖
- **Java 17 或更高版本**
- **Spigot API**
- **OkHttp** 和 **Gson** 库（已包含在插件中）
